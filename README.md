# student-performance-analysis
Student performance analysis using Python, Pandas, NumPy, and visualization libraries such as Matplotlib and Seaborn. The project explores grade distribution, study habits, absences, and gender-based performance differences. It also applies a machine learning model to predict final grades and uncover factors influencing academic success.
